## 💡 Sobre o Projeto

O **Pass Up** é uma solução criada para transformar o engajamento dos usuários em benefícios reais no transporte público. A proposta permite que ações realizadas dentro da plataforma gerem pontos, que podem ser convertidos em créditos para uso em passagens.

O projeto busca unir tecnologia, mobilidade urbana e impacto social, oferecendo uma alternativa simples, segura e acessível para auxiliar usuários no custo diário com transporte.

| | |
|:---:|:---|
| 🎯 **Missão** | Transformar o dia a dia dos usuários do transporte público através de uma plataforma inovadora e sustentável. |
| 🔭 **Visão** | Ser a referência em soluções tecnológicas para o transporte público, promovendo mobilidade e inclusão social. |
| 💎 **Valores** | Inovação · Sustentabilidade · Inclusão · Transparência · Compromisso com o usuário |

**Por que o Pass Up?**

- 🚇 **Mobilidade** — Facilita o acesso ao transporte público por meio de créditos digitais
- 🔒 **Segurança** — Validação por token temporário e limite de resgate por usuário
- 🎮 **Engajamento** — Recompensas por ações na plataforma incentivam o uso contínuo
- 🤝 **Impacto Social** — Contribui para a redução dos custos de transporte e inclusão social
- 🌐 **Futuro** — Expansão para novas cidades e integração com mais sistemas de transporte

## 🖥️ Telas do Sistema

### 🏠 Home
> Página inicial do Pass Up. Apresenta o nome da plataforma, navegação principal e um botão de acesso direto para o login. O fundo com imagem de ônibus reforça o contexto do projeto.

![Home](./screenshot-home.png)

---

### 📖 Sobre
> Explica a proposta do projeto com textos sobre missão, visão e valores, organizados em cards azuis. É a página mais informativa do site.

![Sobre](./screenshot-sobre.png)

---

### 👥 Quem Somos?
> Apresenta cada integrante da equipe com foto, nome, RM e links para GitHub e LinkedIn.

![Quem Somos](./screenshot-quemsomos.png)

---

### ❓ FAQ
> Página de perguntas frequentes com accordion interativo — o usuário clica na pergunta e a resposta aparece ou some. Feito com JavaScript puro.

![FAQ](./screenshot-faq.png)

---

### 📬 Contato
> Lista os canais oficiais de contato da plataforma (e-mail, Instagram, Facebook, LinkedIn) e disponibiliza um formulário com validação de campos e feedback visual ao enviar.

![Contato](./screenshot-contato.png)

---

### 💰 Saldo
> Exibe o saldo atual do usuário em pontos e o equivalente aproximado em reais (100 pontos = R$ 1,00). Também mostra o histórico de ações que geraram pontos, como assistir anúncios e indicar amigos.

![Saldo](./screenshot-saldo.png)

---

### 🎟️ Resgate
> Permite ao usuário informar a quantidade de pontos a resgatar e o cartão de transporte. Após a confirmação, um **token temporário único** é gerado automaticamente via JavaScript, válido por 24 horas, para ser usado em pontos de venda ou aplicativos parceiros.

![Resgate](./screenshot-resgate.png)

## ❓ Perguntas Frequentes

<details>
<summary><strong>Como funciona o resgate de pontos?</strong></summary>
<br>
Para resgatar seus pontos dentro do site da Pass Up, basta clicar em "Saldo" e lá terá sua quantidade de pontos junto da opção de resgate. Informe a quantidade de pontos e seu cartão de transporte para confirmar. Um token temporário será gerado automaticamente.
</details>

<details>
<summary><strong>Quais meios de transporte são aceitos?</strong></summary>
<br>
Os tipos de transportes aceitos são os ônibus intermunicipais e metrôs/trens.
</details>

<details>
<summary><strong>Como assistir os anúncios?</strong></summary>
<br>
Para assistir os anúncios basta entrar na aba "ganho de pontos" e clicar em "assistir anúncio".
</details>

<details>
<summary><strong>Como ver os seus pontos?</strong></summary>
<br>
Para ver seus pontos, basta ir na aba "Saldo". Lá estará seu saldo tanto em pontos quanto em reais.
</details>

<details>
<summary><strong>Existe quantidade máxima de pontos?</strong></summary>
<br>
Não! O Pass Up não possui limite de pontos que o usuário possa ter ou retirar.
</details>

## 🚀 Como Executar

**Pré-requisitos:** ter o projeto salvo no seu computador. Recomendamos usar o **VS Code** com a extensão **Live Server**.

```bash
# 1. Clone o repositório
git clone https://github.com/seu-usuario/pass-up.git

# 2. Abra a pasta no VS Code

# 3. Clique com o botão direito no arquivo index.html
#    e selecione "Open with Live Server"
```

> Sem o Live Server, basta abrir o arquivo `index.html` diretamente no navegador. Não é necessário instalar nenhuma dependência.
